# Milagros Admin
A new Milagros Admin build with
- PHP 8.2+
- Laravel 10
- Jquery 3.7.1
- Tailwind CSS
- DaisyUI

## Set up project
1. Update & install composer
```bash
composer update
```
```bash
composer install
```

2. run npm install
```bash
npm install
```

3. Set key app for laravel
```bash
php artisan key:generate
```

## Run your project
```bash
npm run dev
```
```bash
php artisan serve
```
